ABOUT

Lick it is an experimental arcade game that a player controls a huge physical tongue puppet to accomplish small tasks. So far we have made the first task, which is lick ice cream(on a screen) before it melt.

 

In these project, we explore exaggeration of part of human body that we are not usually think of in everyday life since we can’t see it. What can we see if we are a tongue? What kind of tasks we need to do in everyday life?

 

This game is made in Openframeworks platform with digital hand draw animation. The tongue controller has the feeling like heavy machinery controller, which hacked from playground toy called ‘big dig’ and make it interactive using kinect motion tracking.

 

The project is part of Newline Exhibition at Babycastles NYC
