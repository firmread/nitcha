Lucid : You are dreaming

Cel animation and rotoscope

Role : Director, Designer and Animator

2014

Credits

Designed and directed - Nitcha Tothong

Cast - Udit Mahajan, Shakti, Tharit Tothong,

           Charles Saidel, Karen Chan

Special Thanks - Zac Collopy, Henry, Katherine, Heloise,

 

“Lucid” is an installation that visualize dreams from actual sleep talks sound by using cel animation.

 

Lucid is a state between dream and awake, in the other word, when you are not completely unconscious. “ The concept built up from a question "How would it like if we can imagine what people see in their dream?” Then using sleep talk as an access point to understand dream.
