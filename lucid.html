“Lucid” is an installation that visualize dreams from actual sleep talks sound by using cel animation.

 

Lucid is a state between dream and awake, in the other word, when you are not completely unconscious. “ The concept built up from a question "How would it like if we can imagine what people see in their dream?” Then using sleep talk as an access point to understand dream.

The installation was made a space like a bed room having mattress, pillow, mosquito net for project on it, and headphone for the sound.

 

The animation was projected on mosquito net because first in my own memory mosquito net is always used in bedroom. It contains the feeling of sleeping space. Also it has meaning of protection that when we sleep we need space that safe for us. However, for many of my tester,  said they only use the net when the go camping so people actually has different feeling and memories with this object.

 

Also the way that audiences have to lay down. Their bodies contain memory of the gesture, so many audiences can realize that project feels like dream  with our explaining anything.
