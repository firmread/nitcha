ABOUT

This set is a small project, which all were done in Unity, C# code, modeled in Maya and Sculptris and be a part of Building world class, MFA D+T Parsons The New School for Design.

 

This work is in the theme of New York City. I used 4 symbols that represent the city perfectly, which are taxi, building, trashcan, and rat.

 

I used instantiate code  to clone the original model.
