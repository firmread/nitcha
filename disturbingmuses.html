The Disturbing Muses

Interactive 3D piece

Role : Designer, 3D modeling, C# Code

2014

 

Credits

Special Thanks - Robert Yang

 

“The Disturbing Muses” is a small interactive 3D piece inspired from a documentary "Ways of Seeing” that we can re-contextualize meaning of the picture depends on the way we see it. This piece’s name refers to a painting by Giorgio de Chirico, the Italian metaphysical painter in 1916.

 

The original painting has the mysterious feeling. It has a loose sense of time in a day, place, and era. Even though the painting has harmonious sense, it has contrasting elements in details, such as color and style.

 

This project retains the contrast feelings from the painting but remap them in various ways. First, it allows user to control sunlight so that user knows what time it is in the day. User also has freedom to walk into 3D environment of the painting but still limited to some constraints . Furthermore, user can play and move objects in the painting that makes it more playful than the original, which contrast to uncomfortable industrial sound.
